# Project Roadmap

## Overview

This roadmap outlines the planned development phases for the Retail Analytics Platform.

The project is organized into sprints to reflect how analytics engineering work is planned, built, tested, and delivered.

---

## Sprint 1: Project Setup & Documentation

- Create GitHub repository
- Build professional folder structure
- Write README
- Define business requirements
- Document architecture
- Define core business metrics
- Create project roadmap

---

## Sprint 2: Raw Data & Bronze Layer

- Select retail datasets
- Store raw data files
- Load raw files into Databricks
- Create Bronze tables

---

## Sprint 3: Silver Layer

- Clean raw data
- Standardize column names
- Handle missing values
- Remove duplicates
- Validate data types

---

## Sprint 4: Gold Layer & Data Modeling

- Design star schema
- Create fact tables
- Create dimension tables
- Define table relationships

---

## Sprint 5: Business Metrics

- Build SQL metric calculations
- Create reusable business views
- Validate KPI consistency

---

## Sprint 6: Data Quality Testing

- Add null checks
- Add duplicate checks
- Add referential integrity checks
- Add business rule tests

---

## Sprint 7: Power BI Dashboard

- Connect dashboard to Gold layer
- Build executive summary page
- Build sales analysis page
- Build customer analysis page
- Build returns analysis page

---

## Sprint 8: Final Polish & Release

- Update README with screenshots
- Add architecture diagram
- Review documentation
- Clean repository
- Create Version 1.0 release
