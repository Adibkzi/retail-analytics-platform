# ADR-001: Project Technology Stack

## Status

Accepted

---

## Context

The goal of this project is to demonstrate the skills expected of an Analytics Engineer by building an end-to-end retail analytics platform using modern cloud-based technologies.

---

## Decision

The project will use the following technologies:

- Databricks Community Edition
- SQL
- Python
- PySpark
- Delta Lake
- Power BI
- Git and GitHub

---

## Rationale

### Databricks

Provides a scalable environment for data engineering and analytics workflows using Apache Spark.

### SQL

Used for data transformations, business logic, and analytical queries.

### Python

Supports automation, data processing, and notebook development.

### PySpark

Processes large datasets efficiently within Databricks.

### Delta Lake

Provides reliable storage with ACID transactions and supports the Medallion Architecture.

### Power BI

Creates interactive dashboards for business stakeholders.

### Git & GitHub

Enable version control, collaboration, and project documentation.

---

## Consequences

Using these technologies creates a project that closely resembles a modern Analytics Engineering workflow and demonstrates practical industry skills.
